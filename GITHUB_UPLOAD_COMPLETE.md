### **1. Clone the Repository:**

### **2. Install Dependencies:**
```bash

cd ..
```

### **3. Setup Environment:**
```bash
# Create .env file
NODE_ENV=development
PORT=5000
MONGODB_URI=mongodb://localhost:27017/nsti_college_db
JWT_SECRET=nsti_college_management_jwt_secret_key_2024
```

```bash

```

```bash

npm run dev

# Or start separately

```
